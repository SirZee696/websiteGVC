Median Filter

A Median Filter smooths an image by replacing each pixel's value with the median of its neighbors, preserving edges.

How It Works

The Median Filter is a non-linear digital filter. It slides a kernel over the image, gathers all the pixel values from within the kernel, sorts them, and replaces the center pixel with the median (middle) value.

The Formula

Result(x,y) = median({Image(i,j)})

  • The set includes all neighbors (i, j) within the kernel.
  • This process is repeated for every pixel in the image.

Common Uses

Noise Reduction: Extremely effective at removing salt-and-pepper noise.

Edge Preservation: Unlike mean filters, it smooths noise without blurring sharp edges.

Detail Smoothing: Can reduce minor details and textures while keeping major boundaries intact.

Odd values only (1,3,5,...)
