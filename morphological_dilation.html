Morphological Dilation

Dilation expands bright structures by replacing each pixel with the maximum value under a structuring element. It is used to fill small holes, connect disjoint objects, and thicken shapes.

How It Works

For grayscale images, dilation with structuring element \(S\) computes \[\mathrm{Result}(x,y) = \max\limits_{(i,j) \in S}\, \mathrm{Image}(x+i, y+j).\] For binary images, a pixel becomes white if the structuring element overlaps with any foreground pixel.

Parameters

  • Structuring Element: shape (square, cross, disk)
  • Size: controls radius/half-width
  • Iterations: apply dilation repeatedly

Common Uses

Filling Holes: Close small gaps and holes inside objects.

Connecting Components: Join nearby or broken parts of an object.

Thickening Shapes: Increase the size and prominence of foreground objects.

Structuring Element

Odd values only (1,3,5,...)
