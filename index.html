Image Stuctures

Structuring Element

Learn what a structuring element is and how its shape and size control morphological operations like erosion and dilation in binary and grayscale images.

Launch Interactive

Morphological Erosion

Shrink bright regions by sliding a structuring element across the image. Erosion removes small details and noise, thins shapes, and is foundational for morphological processing.

Launch Interactive

Morphological Dilation

Expand bright regions and fill small holes by sliding a structuring element across the image. Dilation is used to join broken parts, thicken shapes, and is a key operation in morphological image processing.

Launch Interactive

Morphological Opening

Smooth object contours and remove small, isolated bright spots (noise) by performing an erosion followed by a dilation. This operation is ideal for cleaning up binary images without significantly changing the size of the main objects.

Launch Interactive

Morphological Closing

Fill small holes and gaps within foreground objects by performing a dilation followed by an erosion. This operation is useful for joining broken components and smoothing contours from the inside.

Mean Filter

Smooth an image and reduce noise by replacing each pixel's value with the average (mean) of its neighboring pixels. This is a simple and common type of blur filter.

Launch Interactive

Median Filter

A non-linear filter that is highly effective at removing salt-and-pepper noise while preserving edges. It works by replacing each pixel's value with the median value of its neighbors.

Launch Interactive

Gaussian Smoothing

A powerful blurring technique that uses a Gaussian function to average pixels. It's highly effective for reducing noise while preserving edge-like structures better than a simple mean filter.

Launch Interactive

Image Convolution

Apply a kernel to an image to perform operations like blurring, sharpening, and edge detection. Understand the fundamental operation behind many image processing filters.

Unsharp Filter

Sharpen an image by subtracting a blurred version of it from the original. This technique, known as unsharp masking, enhances edges and fine details.

Feature Detection

Robert Cross Detector

A simple edge detection operator that uses a pair of 2x2 convolution kernels to approximate the gradient of an image. It's fast but sensitive to noise.

Launch Interactive

Sobel Edge Detector

A popular edge detection algorithm that uses 3x3 kernels to emphasize horizontal and vertical edges. It's more robust to noise than the Robert Cross operator.

Launch Interactive

Canny Edge Detector

A multi-stage algorithm to detect a wide range of edges. It involves noise reduction, finding gradients, non-maximum suppression, and hysteresis thresholding to produce clean, thin edges.
