RGB Color Space

A geometric interpretation of color.

Each color you see on a screen can be represented as a point in a 3D cube.

The position of the point is determined by the intensity of Green (X), Red (Y), and Blue (Z) light.

Use the sliders to change the color, and rotate the cube to see where that color lives in 3D space.

Hex Code
