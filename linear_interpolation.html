Linear Interpolation (Lerp)

Visualizing the blend between two values.

How it Works

Linear Interpolation, or "Lerp", is a fundamental operation in computer graphics. It finds a value that is some fraction of the way between two other values. We can use it to blend colors, move objects, or change any numerical property over time.

The Formula

Result = A * (1 - t) + B * t

A is the starting value (or color).

B is the ending value (or color).

t is the interpolation factor, a value from 0.0 to 1.0. At t=0, the result is A. At t=1, the result is B.

Live Calculation

R = 0 * (1 - 0.0) + 0 * 0.0 = 0
